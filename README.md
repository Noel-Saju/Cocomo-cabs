# Cab booking app

This is a cab booking app created as part of the software engineering course.



## Technology

This demo uses:

- CometChat Pro 3.0.0
- CometChat UI Kit
- Firebase
- React.js
- Uuid
- Validator
- Leaflet
- Leaflet-geosearch
- Leaflet-routing-machine
- @emotion/core
- dateformat
- emoji-mart
- react-html-parser
- twemoji
